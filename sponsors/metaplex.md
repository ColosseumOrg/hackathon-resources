Metaplex is building the agent exchange for Solana. With the Metaplex Agent Kit, 014 registry, and agent token launchpad, Metaplex is providing the infrastructure for agents to be discoverable and economically viable onchain.

The 014 registry enables developers to register agents onchain as Core NFTs that come with built-in wallets. From there, agents can launch tokens, mint and own NFTs, and participate in agent-to-agent commerce. During Frontier, we encourage teams to build useful agents: production-ready agents with identity, ownership, and economic functionality built in from day one.

- [Register an Agent](https://developers.metaplex.com/agents/register-agent)
- [Run an Agent](https://developers.metaplex.com/agents/run-an-agent)

A simple starting point is to register your agent on Solana using the Metaplex Agent Kit, connect your runtime, and fund the agent wallet so it can begin transacting or deploying programs. From there, builders can launch tokens, coordinate with other agents, or create new agent-driven marketplaces and services.

- [Launch a Token](https://www.metaplex.com/docs/agents/create-agent-token)
- [Metaplex Skill](https://www.metaplex.com/docs/agents/skill)
- [Agent Registry](https://metaplex.com/agents)
- [Agent Kit Docs](https://metaplex.com/docs/agents)

Teams that incorporate identity, discoverability, and economic functionality will be best positioned to build durable agent applications. Metaplex DevRel support and office hours are available for all Frontier hackathon teams.
