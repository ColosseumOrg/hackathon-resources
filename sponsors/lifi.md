LI.FI is universal liquidity infrastructure for Solana. Integrate once to power cross-chain functionality in your app — Solana swaps aggregating Jupiter, DFlow, Titan, OKX, and Fly for best execution, bridging to and from Solana across 20+ bridges and 60+ chains, and seamless cross-chain asset exchanges. A single API integration gives you access to all major DEXs, bridges, and solvers.

- [LI.FI Docs](https://docs.li.fi/)
- [Solana Overview](https://docs.li.fi/li.fi-api/solana)
- [REST API Reference](https://docs.li.fi/api-reference/introduction)

LI.FI also provides tooling for AI agent development — an MCP Server for cross-chain swap functionality, agent skills on GitHub, and an Openclaw skill for bridging and swapping across 35+ chains.
