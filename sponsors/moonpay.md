MoonPay Agents is infrastructure for AI agents to move money. Agents can create wallets, hold assets, and execute on-chain transactions. Agents have access to all MoonPay tools like On/Off-ramps, virtual accounts, commerce, cross-chain swaps, virtual cards, etc.

During Frontier, we'd like to see agents move money both in the real world and agentic economy. We want to see experiences that give agents bank accounts or virtual cards to participate in commerce, see new DeFi experiences using stablecoins, etc.

With one command: `npm install -g @moonpay/cli` and our skills repo, users can get 30+ skills to start building.
