Coinbase Developer Platform gives Frontier teams a practical stack for agentic commerce: x402 for paid APIs, CDP facilitator infrastructure for verification and settlement, Agentic Market for discovering x402-enabled services, onramps/offramps for moving between fiat and crypto, and wallet products for both users and autonomous agents.

x402 is an open payment protocol that revives HTTP 402 Payment Required so APIs, data, tools, and content can charge per request with instant stablecoin payments. Builders can use the CDP facilitator to avoid running their own payment verification and settlement infrastructure across supported networks, including Solana. The x402 MCP server shows how an agent can detect a paid endpoint, sign the payment, retry the request, and return the paid data to an MCP client without manual checkout.

- [x402 Docs](https://docs.cdp.coinbase.com/x402/welcome)
- [x402 MCP Server](https://docs.cdp.coinbase.com/x402/mcp-server)
- [Agentic Market](https://agentic.market/)

For product flows that need funding, onboarding, or custody choices, Coinbase Onramp & Offramp support fiat-to-crypto funding and crypto-to-fiat cash out, while the wallet guide helps teams choose between embedded wallets for app users, server wallets for backend automation, and agentic wallets for AI agents. Coinbase's Agentic Wallet Skills package gives agents prebuilt actions for authentication, funding, sending USDC, trading, finding x402 services, paying for services, and monetizing APIs.
