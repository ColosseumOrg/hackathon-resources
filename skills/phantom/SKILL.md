# Phantom Skills

Full Phantom SKILL.md files are available in the [Colosseum Copilot repo](https://github.com/ColosseumOrg/colosseum-copilot/pull/5).

Phantom provides 4 skills for Frontier builders:

- **phantom** — Router skill that directs to the right Phantom product
- **phantom-connect** — Embedded wallet for user-facing dApps
- **phantom-mcp-server** — Agent wallet via MCP
- **phantom-cash** — CASH stablecoin for payments and payouts

Install via: `npx skills add ColosseumOrg/colosseum-copilot`
